# Task Management System  

## Installation

First Run this command to start the application.

```
php artisan start:app
```

Then run this command to update statistics .

```
php artisan queue:work
```

then run tests.

```
php artisan test
```

Then go to 

```
 http://127.0.0.1:8000 
```
